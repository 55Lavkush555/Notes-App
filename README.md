# 📝 Notes App

A simple and powerful Notes App built using React. This app allows users to create, edit, delete, search, and view notes easily.

## 🚀 Live Demo

👉 [Click here to use the app](https://notes-app-kappa-sandy.vercel.app/)

---

## 📌 Features

* ➕ Add new notes
* ✏️ Edit existing notes
* 🗑️ Delete notes
* 🔍 Search notes in real-time
* 📖 View full note content
* 💾 Data stored in localStorage

---

## 🛠️ Tech Stack

* React.js
* React Router DOM
* JavaScript (ES6+)
* CSS

---

```
src/

 │    ├── NewNote.jsx
 │    ├── EditNote.jsx
 │    └── Note.jsx
 ├── App.jsx
 └── App.css
```

---

## ⚙️ How It Works

* Notes are stored in **localStorage**
* Each note contains:

  * Title
  * Content
* Users can:

  * Create a note
  * Edit it anytime
  * Delete unwanted notes
  * Search notes instantly
  * Click to view full note
